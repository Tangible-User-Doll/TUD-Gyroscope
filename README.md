From ESP to GY521(right hand):  
GND -> GND  
3V3 -> VCC  
D1(SCL) -> SCL  
D2(SDA) -> SDA

From ESP to GY521(left hand):  
GND -> GND  
3V3 -> VCC  
D1(SCL) -> SCL  
D2(SDA) -> SDA  
3V3 -> AD0

From ESP to rgb led (pins 1-4, from left to right):  
GND -> 2  
D6 -> 1  
D7 -> 3  
D8 -> 4
